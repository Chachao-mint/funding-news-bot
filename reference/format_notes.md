# 過去記事から学習したフォーマット（mintvc Substack）

参考記事3本（513-2 / 512-4 / 511-3）を分析した結果。記事生成時はこの型に合わせる。

## 記事作成手順（運用者から共有された方法）
1. Substackのマイページにアクセスする。
2. ホーム画面から、過去に投稿した資金調達ニュースを Duplicate（複製）する。
3. 複製した過去記事の中身を、当日PR TIMESで調べた内容に差し替える。

ポイント: 記事はゼロから書き起こすのではなく、過去記事を複製して中身を入れ替える方式。
そのため、過去記事と全く同じレイアウト・体裁を保ったまま、各社セクションの
情報（企業名・画像・タイトル引用・URL・引受先/融資先）だけを当日分に置き換える。

## 記事タイトル
- 形式: `M/D(曜日) N件の資金調達ニュース`
- 例: `5/13(水) 2件の資金調達ニュース`

## 構成（上から順）
1. タイトル
2. リード文なし。タイトル直後に企業名の番号リストを置く。
   ```
   1/ アナウト株式会社
   2/ Check Inn株式会社
   ```
3. 各社の詳細セクション（下記フォーマット）。セクション間は `* * *` で区切る。
4. 末尾の締め。

## 各社セクションのフォーマット
記載順序:
1. 番号＋企業名（例: `①ライトタッチテクノロジー株式会社`）
2. アイキャッチ画像 — PR TIMESのプレスリリース冒頭の画像をそのまま使う
3. 画像下の文章 — PR TIMESのプレスリリースのタイトルを「一字一句そのまま」引用する。
   要約や言い換えはしない。
   - 例: `ライトタッチテクノロジー株式会社、採血不要の血糖値センサー開発へ──５億円追加調達でグローバル展開を加速`
4. ソースURL（PR TIMES）— `https://prtimes.jp/...`
5. `▼引受先` — エクイティ調達の投資家名を列挙
6. `▼融資先` — デット調達の銀行名を列挙（デット案件のみ）

注意: 1案件でエクイティとデットの両方を含むケースがある。
その場合は `▼引受先` と `▼融資先` を両方併記する。
（例: ライトタッチテクノロジーは投資家4社からの増資＋紀陽銀行からの融資）

## トーン・体裁
- 簡潔。記事全体で200語前後と短め。
- 名詞中心の箇条書きスタイル。です/ます調は最小限（記事により多少のゆれあり）。
- 絵文字は使わない。ニュートラルな報道トーン。

## 末尾の締め
- Xアカウント（@Chayano0810）への誘導と購読促進。
- 例: `#今日の資金調達ニュース で検索するとXでも同じ内容を投稿しています！`
