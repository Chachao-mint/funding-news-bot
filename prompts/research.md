# リサーチプロンプト（research.md）

このファイルは、平日18時の処理フローのうち
「ステップ1 リサーチ」「ステップ2 絞り込み」「ステップ3 裏取り」で従う指示文です。

## ゴール
当日PR TIMESに出た、日本のスタートアップの資金調達ニュースをすべて見つけ、
記事にする対象を確定し、各案件の事実を裏取りする。

## 手順

### 1. PR TIMESで検索
- PR TIMES（https://prtimes.jp/）で `config.json` の `search.keywords` の各キーワードを検索する。
- 当日（実行日）に配信されたプレスリリースに絞る。前日以前のものは対象外。

### 2. 絞り込み
次の条件をすべて満たすニュースだけを対象にする:
- 資金調達のニュースである（エクイティ調達 または デット調達）。
- 発表企業が日本のスタートアップである。海外企業は除外。
- `config.json` の `filter.exclude` に該当しない
  （上場企業のIR / M&A・買収のみ / 助成金・補助金のみ / クラウドファンディングのみ は除外）。

### 3. 裏取り（各案件ごと）
対象の各案件について、元のPR TIMESプレスリリースを開き、以下を正確に確認する:
- 企業名（正式名称）
- プレスリリースのタイトル（一字一句そのまま）
- 調達額
- 調達ラウンド（シリーズA/B、シード、ポストシード 等）
- 引受先（出資した投資家名）
- 融資先（融資した銀行名。デット調達の場合）
- アイキャッチ画像（プレスリリース冒頭の画像）
- ソースURL（PR TIMESのページURL）

確認できない情報は「不明」と記録し、推測で埋めない。

## アウトプット
確認した内容を案件ごとに整理し、`output/today/` に当日のリサーチメモとして保存する。
このメモを `article.md`（記事作成）と `x_draft.md`（X下書き）の入力として使う。
